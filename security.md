# Security

## Intended Users
The intended users of this repository are the course instructor, TAs, and me.

## Security Risks
If the code or data in this repository fell into the wrong hands, the main risk would be academic integrity issues such as other students copying the code. There is no sensitive personal data or credentials stored in the repository.

## Security Measures
This repository does not contain sensitive data or secrets. No API keys or passwords are stored in the repo. The repository is only used for coursework and standard GitHub permissions control access.
